# GeoGuess scripts



## Installation

```bash
npm install -g @geoguess/geoguess-scripts
```

### Clean rooms

```bash
clean-rooms -f <file-path> -c <database>

Options :
  -f, --file-path    Path to the file to be uploaded                    [string] [default : "./keys.json"]
  -d, --databaseUrl  Realtime database url                             [string]
  -h, --help        
```
